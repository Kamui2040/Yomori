# Development APKs

Yomori development APKs are built by GitHub Actions and can be downloaded directly on a phone.

The development package uses `io.github.kamui2040.yomori.debug`, a dedicated public test certificate, and filenames containing the app version, commit count, short commit SHA, and ABI.

The public development certificate is intentionally not suitable for a production release. A future production build will use a separate protected signing key.
