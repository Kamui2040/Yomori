# Support

Yomori is in early development and has no public production release.

## Supported issue scope

Use the repository issue templates for reproducible problems involving:

- CBL import, parsing, persistence, matching, or review;
- reading-list progress and cross-series reader behavior;
- Yomori UI, accessibility, backup, restore, build, or installation;
- extension loading, trust, signature, or compatibility behavior caused by Yomori;
- privacy, permission, or network behavior caused by Yomori.

## Out of scope

Yomori does not provide support for:

- finding or recommending content sources;
- choosing an extension repository;
- third-party source availability, credentials, scraping, rate limits, or content;
- extension-specific bugs that do not originate in Yomori;
- copyrighted content requests.

## Before reporting

- Search existing issues.
- Include the Yomori version or commit, package ID, Android version, and device.
- Provide minimal reproduction steps.
- Redact credentials, cookies, tokens, private CBL data, and copyrighted pages.

Security vulnerabilities must use the private route in `SECURITY.md`, not a public issue.
